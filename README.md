# 🚀 Seek Local Tunnel (SLT) - Network Dashboard

**Seek Local Tunnel** is a modern desktop application designed to simplify tunneling your local machine to the public internet using `localtunnel`, while also serving as an integrated Control Panel for XAMPP services.

---

## ⚠️ System Prerequisites (Important)

Although this application is provided as an `.exe`, your system still requires certain "engines" for its features to work perfectly:

1. **Node.js & NPM**: Must be installed so the `npx` command can execute `localtunnel`. [Download Node.js here](https://nodejs.org/).
2. **XAMPP**: Must be installed in the default directory (`C:\xampp\`) for the Start/Stop Service feature to detect system files.
3. **Internet Connection**: Required for the handshake process between your local machine and the public tunnel server.

---

## ✨ Key Features
* **One-Click Tunneling**: Instantly expose localhost to a public URL with a custom subdomain.
* **XAMPP Service Manager**: Control Apache, MySQL, FileZilla, Mercury, and Tomcat services directly from the dashboard without opening the native panel.
* **System Monitor**: Monitor CPU and RAM usage in real-time to maintain local server stability.
* **Authentication System**: Local Login & Register system to ensure only you have access to the dashboard.
* **Clean UI**: A responsive, minimalist, and distraction-free modern interface.
* **Live Log System**: Transparently monitor connection activities, generated URLs, and error messages.

---

## 🛠️ Technology Stack
This application is built using the **Eel** architecture, combining the power of **Python** for system control and **Web Technologies** (HTML/CSS/JS) for a modern user interface.

---

## 📖 How to Use

1. **Preparation**: Ensure Node.js and XAMPP (installed at C:\xampp) are ready on your computer.
2. **Launch**: Open the `Seek Local Tunnel.exe` file.
3. **Authentication**: Please Register (if it's your first time) and then Login to access the dashboard.
4. **Control XAMPP**: Activate the required services (e.g., Apache & MySQL) through the XAMPP tab within the app.
5. **Activate Tunnel**: Enter your Port (e.g., 80) and your chosen Subdomain, then click **Establish Connection**.
6. **Done**: Wait for the log to display the URL `https://...`. Your local project is now online and accessible to anyone!
